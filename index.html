PROJECTS

The Student Learning Management System (SLMS) is a comprehensive web-based platform designed to enhance the learning experience for students, faculty, and administrators. This system facilitates efficient course management, seamless student-faculty interaction, and structured access to study materials. Developed using Java Full Stack, the project integrates Spring Boot for the backend, MySQL for data storage, and JSP for dynamic web pages, ensuring a robust and scalable solution.

The SLMS includes three main modules:

  • Admin Module: Enables the management of students, faculty, and courses, along with monitoring system performance.
  • Student Module: Provides students with a personalized dashboard to access study materials, track enrolled courses, and manage assignments.
  • Faculty Module: Allows faculty members to upload course materials, assess student progress, and interact with students effectively.

This system aims to streamline academic processes, making education more accessible, structured, and engaging for both students and faculty.



Student Learning Management System

A full-stack Java project built using Spring Boot, MySQL, and JSP for the front end. The system includes three modules: Admin, Student, and Faculty.

Admin Module: Handles student and faculty registration, course management, and profile updates.

Student Module: Allows students to view enrolled courses and access study materials.

Faculty Module: Enables faculty to upload study materials and manage student progress.

Student Login Interface

The student login page is designed with a clean and intuitive interface. Students can log in using their credentials to access their dashboard.

Student Dashboard

Enrolled Courses
